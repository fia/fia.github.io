# 上传图片人脸识别小游戏 [face-name]

## 简介

通过 [*eyekey*](http://eyekey.com/) 提供的API，实现照片面部识别，制作成创意小游戏。

> DEMO 为简单版功能实现，实际上线版本结合了微信API，包含了微信图片上传，微信分享等具体社交功能
